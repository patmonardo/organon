# Attention Reports

Attention reports are user-facing inspection records for feature evaluation.

---

## Doctrine

An attention report tells the user what the system considered important during a feature evaluation. It is part of interpretability, not UI ornament.

---

## Should Include

- dataset name
- feature specification
- evaluation mode
- selected rows or spans
- summary of matches or salience
- provenance

---

## Review Rule

An attention report should help a user decide whether the feature evaluation deserves trust.
