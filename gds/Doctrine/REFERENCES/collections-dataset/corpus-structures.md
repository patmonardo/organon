# Corpus Structures

A Corpus is a source organized as documents, spans, annotations, and provenance.

---

## Doctrine

Corpus is the first linguistic source object. It is not raw text. It is source material prepared for Observation.

---

## Core Parts

- document identity
- source reference
- text or payload
- spans
- annotations
- provenance

---

## Review Rule

A corpus example should make document identity and source provenance visible. If text enters without a stable source record, downstream marks cannot be trusted.
