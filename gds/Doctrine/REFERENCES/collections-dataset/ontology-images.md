# Ontology Images

An ontology image is the realized table family that records models, features, constraints, queries, and provenance.

---

## Doctrine

Ontology images are not the ontology itself. They are runtime-readable images of semantic intent.

---

## Tables

- models
- features
- constraints
- queries
- provenance

---

## Review Rule

An ontology image should always answer: which program created this, what model does it represent, and which features were realized?
