# Dataset Standard Library Resources

Dataset stdlib resources are curated source packages for scientific workflows.

---

## Doctrine

A stdlib resource is not just a download. It is a named source commitment. It lets the platform say: this corpus, archive, or reference material is reusable, fetchable, and traceable.

---

## Resource Report

A resource fetch should report:

- resource name
- archive path
- data directory
- byte count
- error state if fetch failed

This report becomes the first source trace for later workflows.

---

## Position in the Arc

Stdlib resources live at Stage 1: Source.

They prepare material for Observation. They do not parse, mark, reflect, or judge by themselves.

---

## Review Rule

When adding a resource, ensure:

1. Stable name
2. Reproducible fetch location
3. Clear local materialization path
4. No hidden semantic interpretation during fetch
