# Feature Structures

Feature structures are typed symbolic records used for semantic marking and unification.

---

## Doctrine

Feature structures are the algebraic bridge from observed structure to essence preparation. They let a program state what roles a candidate object carries before runtime execution.

---

## Duties

A feature structure should represent:

- named features
- typed values
- variables
- reentrance
- unification constraints

---

## Review Rule

Do not collapse feature structures into anonymous JSON blobs. They are first-class symbolic objects.
