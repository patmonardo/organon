# Unification

Unification is the operation that merges compatible feature structures and rejects incompatible ones.

---

## Doctrine

Unification prepares essence. It determines whether separate marks and roles can live in the same semantic object.

---

## Result

A unification result should record:

- success or failure
- resolved feature values
- conflicting paths if failure
- provenance of both inputs

---

## Review Rule

Unification failure is not an error to hide. It is semantic information that may feed contradiction or Principle evaluation.
