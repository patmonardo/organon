# Relative Reflection

Relative Reflection names the first six moments by which appearance becomes essence-articulated before Absolute Reflection.

---

## Moments

1. Being
2. Identity
3. Opposition
4. Contradiction
5. Ground as condition
6. Ground evaluated

These moments are relative because each one depends on the preceding determinations and prepares the next.

---

## Doctrine

Relative Reflection prepares. It does not yet name the Concept. It articulates the entity enough that Absolute Reflection can hold all moments together at the threshold.

---

## Review Rule

A Relative Reflection artifact should preserve ordered moments. If the order is lost, the reflective process is no longer readable.
