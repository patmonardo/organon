# Kantian Synthesis and Observation

Observation is not passive reception. It is synthesis: the system organizes what is given so that it can become an object of knowledge.

---

## Doctrine

In the Knowledge Agent arc, Kantian synthesis appears in Source -> Observation. The source is given; Observation retains, derives, classifies, parses, and structures it.

## Review Rule

Do not confuse Observation with Concept. Observation makes appearance available. Reflection and Principle must still occur before Concept emerges.
