Span:

- sourceFile: `src/logos/ys/ys_iv_17.ts`
- lineStart: 129
- lineEnd: 180

Summary:

Key points: (KeyPoint)

- k3. Achieving this clarity requires absolute discipline; lacking that _upārāga_/attunement, the object (the genetic truth) remains unknown to the philosopher.
