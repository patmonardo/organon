# Legacy Scripts

This directory contains scripts that are still useful in some workflows but are not part of the active minimal extraction path.

Current core path uses:
- `download_corpus.py`
- `extract_gita_records.py`
- `extract_bs_records.py`

Legacy scripts retained here:
- `extract_semantic_index.py`: HTML-wide node/link graph extraction.
- `qa_translation_records.py`: QA for translation passage-record schema.
- `project_records_to_ys.py`: projection of curated passage records into TypeScript stubs.

Move scripts back to `ref/sankara/scripts/` if they become active again.
