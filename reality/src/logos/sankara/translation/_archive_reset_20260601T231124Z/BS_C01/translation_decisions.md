# Translation Decisions

1. Draft records are source-faithful and interpretation-minimal.
2. Translation fields remain placeholders until manual philological pass.
3. Passage ordering follows first appearance in BS root witness HTML.
4. Record IDs use section-based numbering for deterministic review sequencing.

## Basic Translation Target (Pilot)

1. Keep two distinct translation layers for each passage: literal first, technical second.
2. Preserve doctrinal force words directly in lexical notes rather than smoothing them into paraphrase.
3. Treat opening sutra particles (for example atha/atah) as semantically active, not ornamental.
4. Prefer reversible translations: an informed reader should recover the source grammatical pressure.
5. Move interpretation into interpretive_note only; avoid embedding it in literal_translation.
