### Source Text
छन्दोभिधानान्नेति चेन्न तथा चेतोर्पणनिगदात्तथाहि दर्शनम् ॥२५॥

### Transliteration (IAST)
chandobhidhānānneti cenna tathā cetorpaṇanigadāttathāhi darśanam ||25||

## Translation

### Literal
If it be said 'No, because the meter (chandas) is named,' no - because scripture states fixation of mind in that way; for so indeed it is seen. (25)

### Technical
Sutra 1.1.25: chandas-reference does not block Brahman-reference; Gayatri serves as support for contemplative ceto-arpana toward Brahman.

### Interpretive Note
Sutra preserves symbolic-meter language while retaining Brahmanic purport.
